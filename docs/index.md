# 文档索引

本文档用于统一管理项目中的设计、计划与报告类文档，便于后续维护、查阅和持续迭代。

## 设计类
- [项目设计文档](design/项目设计文档.md)：说明项目背景、架构设计、模块划分、成本控制与风险兜底方案。

## 计划类
- [第一批PR拆分计划](plan/第一批PR拆分计划.md)：说明第一批 PR 的拆分原则、执行顺序与验证方式。

## 报告类
- [项目介绍](report/项目介绍.md)：赛题原始说明与项目立题依据。

## 使用说明

- 设计类文档优先用于确认整体方案与开发方向。
- 计划类文档优先用于推进阶段性开发与 PR 拆分。
- 报告类文档用于保留赛事原始材料与说明依据。
